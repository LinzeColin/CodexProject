# EVA_OS 给使用者的极简说明

EVA_OS 是个人研究与决策支持中台，不是自动交易软件。

你可以用它做：

- 量化策略研究和回测；
- 参数扫描和报告生成；
- 证据整理和数据质量检查；
- Token ROI、现金流、政策、消费等研究台账；
- 总控驾驶舱状态检查；
- 后续接入更多只读参考数据。

你不能把它当成：

- 自动下单软件；
- 投资建议系统；
- 自动付款/税务/政府提交工具；
- 保存交易密码或账户隐私的工具。

日常建议：

1. 打开 `EVA_OS.app`。
2. 先看总控驾驶舱。
3. 缺证据的结果不要用于决策。
4. 回测和报告只作为研究参考。
5. 重要结论需要人工复核。
